### Neovim config
NeoVim config for JavaScript/TypeScript and Rust and C and nix development.
## Installation
- Download neovim using package manager
- Clone repo
- Install configs using linux `cp` command
  ```cp -rf /dist/to/nvim-config/* ~/.config/nvim```
- Run `:MasonInstallAll`
- Enjoy!!!
